Editor Guidelines

Everything you need to know for managing Your Bliss in 716 content.

Events

  • Adding New Events (Using AI):
    • To generate events quickly, use ChatGPT or any AI tool with the following prompt:
      • Get me a list of events in and around buffalo. For the events around buffalo, make sure they are less than a 30 minute drive. Additionally, for each event, follow the structure below:
      1. Name of the event.
      2. One line description of the event.
      3. The month the event is available from
      4. General time of the event (example: 11:00 AM - 3:00 PM)
      5. Name of the venue and the location of the venue (example: Delaware Park, Buffalo, NY)
      6. The official website link (url) of the event where tickets can be booked (if the event is free, still get the official website)
      IMPORTANT: Give me about 15-20 events starting from this month and give a higher concentration of events that are in the current and the next 2 months.
    • Always verify links provided by AI — sometimes they may be incorrect or non-functional.
    • In the above prompt, you can adjust the number of events based on current needs.
  • Click Events → Create to add a new event.
  • Name: The full event name, e.g. Buffalo Holiday Spring Market
  • Image: Paste a valid image URL from reliable sources such as Unsplash, Pexels, Pixabay or similar stock image websites.
  • URL: Paste the official website link of the event. Always ensure the link is accurate and working.
  • Month Format: Use one of the following standardized formats for the month field:
    • Month – for single-month events (e.g. May)
    • Month–Month – for events spanning multiple months (e.g. May–August)
    • Year-Round – for events available throughout the year
    • Monthly – for events that occur every month
    Ensure consistency in formatting across all events.
  • Badges should be separated by commas (e.g. Food, Local vendors, Music).

Digital Magazines

  • Click Magazines → Create to add a new issue.
  • Name: The full issue name, e.g. March 2026
  • Date Added: Displayed on the website under the cover, e.g. March 2026
  • Cover Image (Upload): Click "Choose File" and select your cover image from your computer. It will be saved automatically to assets/magazines/images/
  • PDF File (Upload): Click "Choose File" and select the magazine PDF. It will be saved automatically to assets/magazines/media/
  • The newest issue always appears first — no sorting needed.

Wellness Articles

  • Set Category correctly: health, wealth, spirit, or happiness — this controls which section the article appears in.
  • Enable Featured to show the article in the homepage spotlight grid.
  • Minutes: Estimated reading time in minutes (e.g. 5).
  • Use Sort Order (lower number = appears first) to control the listing order.
  • Image: You have two options for the article image:
    • Enter a URL / Path (e.g., from Unsplash or a manual path like assets/images/name.jpg).
    • Or Upload a file from your computer which will be saved to assets/images/. Choosing a file overrides the URL field.

Recipes

  • Enable Featured to highlight the recipe in the "This Month's Top Picks" slider on the Recipes page.
  • Use Sort Order (lower = first) to control which recipes appear at the top of the main listing.
  • Category: Select up to 3 relevant tags (e.g. breakfast, vegan, dinner).

Restaurants

  • Catchy Phrase: A short tagline displayed under the restaurant name, e.g. Hidden Oasis.
  • Description: The full article-style text. Separate paragraphs with a blank line.
  • Use Sort Order (lower = first) to control which restaurants appear first on the page.

Attractions

  • Category: Select up to 2 tags — these drive the filter buttons on the Attractions page (e.g. museums, parks & nature).
  • Enable Featured to show the attraction in the "Must-See Attractions" spotlight slider.
  • Price: Use Free or an amount like $15/person.
  • Hours: Use a clear format, e.g. Mon–Fri: 9am–5pm.
💡 Pro Tip: Always use high-quality images to maintain the premium look of the Your Bliss brand. Recommended ratios: 16:9 for article/event/recipe cards, 3:4 for magazine covers.
